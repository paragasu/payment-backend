# payment-backend
nodejs payment backend using paypal and braintree api


#Installation
```
  $git clone git@github.com:paragasu/payment-backend.git
  $npm install
  $npm run test
  $npm run dev
```
Open browser and point to [http://localhost:3000](http://localhost:3000)
